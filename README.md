# Project Dossier

Project Dossier es una plataforma tipo SaaS diseñada para automatizar la preparación de reuniones mediante la generación automática de dossiers (informes) sobre los participantes.

## 🧠 Problema

Actualmente, los profesionales invierten una cantidad significativa de tiempo investigando a las personas con las que se reunirán, buscando información en múltiples fuentes como LinkedIn, noticias y registros empresariales. Este proceso es manual, repetitivo e ineficiente.

## 💡 Solución

Project Dossier automatiza este proceso mediante:

- Integración con calendarios (Google Calendar y Outlook)
- Identificación automática de participantes en reuniones
- Recolección de información desde múltiples fuentes externas
- Procesamiento y análisis de datos utilizando inteligencia artificial
- Generación de un dossier completo antes de cada reunión

## ⚙️ Funcionalidades

- 📅 Integración con calendarios
- 🤖 Generación automática de dossiers
- 📊 Dashboard de control y seguimiento
- 🔍 Búsqueda inteligente de expedientes
- 🧠 Análisis y resumen mediante IA
- ⚙️ Reglas de automatización personalizadas

## 🏗️ Arquitectura

- **Frontend:** Next.js
- **Backend:** FastAPI (Python)
- **Base de datos:** PostgreSQL
- **Cache:** Redis
- **IA:** Modelos de lenguaje (LLM)
- **APIs externas:** datos empresariales, noticias y perfiles profesionales

## 📊 Alcance del proyecto

Este proyecto fue desarrollado en el contexto de la asignatura Gestión de Proyectos Informáticos (GPI), integrando tanto desarrollo de software como metodologías de gestión de proyectos, incluyendo:

- Estructura de desglose del trabajo (WBS)
- Cronograma y diagrama de Gantt
- Gestión del valor ganado (EVM)
- Análisis de Curva S
